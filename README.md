# DroidTales Blog

A modern, responsive Jekyll blog powered by the **Chirpy theme** and hosted on **GitHub Pages**.

## 🚀 Quick Start

### Local Development

```bash
# Clone the repository
git clone https://github.com/droidtales/droidtales.github.io
cd droidtales.github.io

# Install dependencies
bundle install

# Serve locally
bundle exec jekyll serve

# Visit http://localhost:4000
```

### Write a New Post

Create a file in `_posts/` with the format: `YYYY-MM-DD-title.md`

```markdown
---
title: Your Post Title
date: 2026-05-13 10:00:00 +0000
categories: [Category]
tags: [tag1, tag2]
---

# Your content here
```

Then push to main:

```bash
git add .
git commit -m "Add new post: Your Post Title"
git push origin main
```
